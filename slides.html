Smallest Useful Call

The API starts as one Python call.

from openai import OpenAI

client = OpenAI()

response = client.responses.create(
    model="gpt-5.5",
    input="Give me one hackathon app idea for improving local events."
)

print(response.output_text)
Sample result: EventSpark A tiny app that turns a neighborhood event theme into a poster concept, invite copy, and a three-step launch checklist. Why this matters: one API call becomes a product behavior you can demo.
Source: OpenAI API Quickstart and Text generation docs04 / 09

Beyond Text

The API is a stack of product capabilities.

Text
Summaries, plans, classification, structured JSON, app logic.
Images
Generate posters, avatars, storyboards, and product visuals.
Speech to text
Transcripts, captions, meeting notes, searchable audio.
Text to speech
Narration, accessibility, generated voice responses.
Realtime
Low-latency voice agents and live audio/text sessions.
Tools
Connect models to APIs, actions, retrieval, and app state.
Fast demo ideas: Images -> generate a team poster. Speech to text -> transcribe judge feedback. Realtime -> make a voice tutor. Tools -> let the model create tickets, files, or calendar events.
Pick the capability that makes the demo feel alive07 / 09
