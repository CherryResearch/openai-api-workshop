Smallest Useful Call

The API starts as one Python call.

from openai import OpenAI

client = OpenAI()

response = client.responses.create(
    model="gpt-5.5",
    input="Give me one hackathon app idea for improving local events."
)

print(response.output_text)
Source: OpenAI API Quickstart and Text generation docs03 / 07
Codex Prompt

Ask for behavior. Let Codex assemble the files.

Create a Python CLI called hack_idea.py. It should take a theme argument and call the OpenAI Responses API. Return a project idea, user story, build steps, and demo script.
The important move: you steer the product while Codex handles implementation, execution, and repair.

Beyond Text

The API is a stack of product capabilities.

Text
Summaries, plans, classification, structured JSON, app logic.
Images
Generate posters, avatars, storyboards, and product visuals.
Speech to text
Transcripts, captions, meeting notes, searchable audio.
Text to speech
Narration, accessibility, generated voice responses.
Realtime
Low-latency voice agents and live audio/text sessions.
Tools
Connect models to APIs, actions, retrieval, and app state.
Pick the capability that makes the demo feel alive06 / 07
